# python_basics / Python 基础练习

这个目录用于记录 Python 工程基础阶段的小项目练习.  
This directory records small project practices in the Python engineering fundamentals stage.

当前阶段的目标不是追求复杂算法, 而是训练基础代码能力, 文件处理能力, 命令行使用能力和项目组织能力.  
The goal of this stage is not to pursue complex algorithms, but to practice basic coding skills, file processing, command-line usage, and project organization.

## 项目列表 / Project List

### 1. text_statistics_tool / 文本统计工具

位置: `text_statistics_tool/`  
Location: `text_statistics_tool/`

用于读取文本文件并统计字符数, 单词数, 唯一单词数和高频词.  
Used to read text files and count characters, words, unique words, and frequent words.

### 2. file_extension_counter / 文件扩展名统计工具

位置: `file_extension_counter/`  
Location: `file_extension_counter/`

用于统计指定文件夹中不同文件扩展名的数量.  
Used to count different file extensions in a specified folder.

### 3. csv_score_analyzer / CSV 成绩分析工具

位置: `csv_score_analyzer/`  
Location: `csv_score_analyzer/`

用于读取 CSV 文件并统计学生成绩信息.  
Used to read CSV files and analyze student score information.

### 4. log_keyword_counter / 日志关键词统计工具

位置: `log_keyword_counter/`  
Location: `log_keyword_counter/`

用于统计日志文件中指定关键词出现的次数.  
Used to count occurrences of a specified keyword in log files.
